# ChatGPT写简历的100条神级提示词

> 别再对着空白文档发愁了！复制粘贴这些提示词，AI帮你写出专业简历。

## 一、简历基础构建（1-20）

### 个人简介/Self Summary
```
1. "根据以下信息帮我写一段专业的个人简介：[姓名/学历/年限/核心技术]，要求简洁有力，控制在3行以内"
```
```
2. "我是一个[X年经验的][岗位]从业者，帮我写一个吸引HR的个人标签（一句话），突出我的核心竞争力"
```
```
3. "帮我重写这段自我介绍：[原文]，让它更符合[目标行业]的专业风格"
```

### 工作经历优化
```
4. "用STAR法则重新组织这段工作经历：[原始描述]。要求有具体数字和可量化的结果"
```
```
5. "我把这段工作经历翻译成英文：[中文内容]。要求用词地道、符合外企简历习惯"
```
```
6. "我有以下工作经历，帮我提炼出3个最能体现[目标岗位]能力的亮点：[详细经历]"
```

### 项目经验描述
```
7. "帮我把这个项目经历写得更有技术深度：[简单描述]。要求体现技术挑战、解决方案、最终效果"
```
```
8. "这是一个[项目类型]的项目，帮我写出结构清晰的项目描述，包括：背景、职责、技术栈、成果"
```
```
9. "我要申请[目标岗位]，帮我从这些项目中挑选最相关的2-3个，并重新包装描述：[所有项目列表]"
```

### 技能关键词
```
10. "根据这个[目标岗位]的JD，帮我列出应该出现在简历中的技术关键词，按重要程度排序"
```
```
11. "帮我评估这份简历的技术关键词密度是否足够：[简历内容]。给出改进建议"
```

## 二、求职信/Cover Letter（21-35）

```
12. "根据我的简历和这家公司的[岗位名称]，写一封求职信。要求：展现我对公司和岗位的了解，说明为什么我合适"
```
```
13. "写一封简短的求职邮件（用于LinkedIn私信），控制在150字以内，自然不推销感"
```
```
14. "我转行从[A行业]到[B行业]，帮我写一封解释转行动机的求职信，强调可迁移技能"
```
```
15. "这是一家[公司类型/文化特点]的公司，帮我调整求职信的语气和风格"
```

## 三、面试准备（36-60）

### 自我介绍
```
16. "帮我准备一个2分钟的自我介绍版本，适用于[目标岗位]面试。结构：我是谁→我能做什么→为什么选择这里"
```
```
17. "准备一个30秒的电梯演讲版自我介绍，用于招聘会或社交场合"
```

### 行为面试（STAR）
```
18. "预测[目标岗位]可能会问的5个行为面试题，并给我STAR格式的回答模板"
```
```
19. "帮我把这个经历改写成完美的行为面试回答：[某次解决困难/冲突/挑战的经历]"
```

### 技术面试
```
20. "我是[岗位]候选人，列出最可能被问到的技术问题（按频率排序），并给出答案要点"
```
```
21. "帮我准备一个'最大的缺点是什么'的回答策略，既要真实又不能致命"
```

### 反问环节
```
22. "面试结束时面试官说'你有什么问题问我吗'，帮我准备3个有水平的问题，体现我的思考深度"
```
